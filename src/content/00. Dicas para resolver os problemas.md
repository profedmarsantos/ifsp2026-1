# **DICA DE ROTEIRO PARA RESOLVER ALGORITMOS EM LINGUAGEM C**

## **Ordem fixa (seguir sempre):**

1. Entender o problema
2. Definir variáveis
3. Ler dados
4. Processar
5. Mostrar resultado
6. Finalizar

---

# **ROTEIRO PRINCIPAL (USAR SEMPRE)**

## **1. Entender o problema**

**Pergunta:**

* Qual deverá ser o resultado?

**Ação:**

* Ler o enunciado
* Identificar exatamente o que precisa ser mostrado no final

---

## **2. Definir variáveis**

**Pergunta:**

* Quais variáveis eu preciso usar?

**Ação:**

* Criar as variáveis necessárias

---

### **REGRA (tipo da variável):**

* Use `int` → para números inteiros
* Use `float` → para números com decimal

---

## **3. Ler dados**

**Pergunta:**

* Quais valores o usuário precisa digitar?

**Ação:**

* Ler todos os valores

---

### **REGRA (sempre usar esse padrão):**

```c
printf("Digite ...: ");
scanf("formato", &variavel);
```

---

### **REGRA (formato):**

* `%d` → int
* `%f` → float

---

## **4. Processar**

**Pergunta:**

* Que cálculo eu preciso fazer?

**Ação:**

* Fazer o(s) cálculo(s) necessário(s)

---

## **5. Mostrar resultado**

**Pergunta:**

* O que eu preciso mostrar?

**Ação:**

* Mostrar o(s) resultado(s)

---

### **REGRA (sempre usar esse padrão):**

```c
printf("Texto: formato", variavel);
```

---

### **REGRA (formato):**

* `%d` → int
* `%f` → float

---

## **6. Finalizar**

**Pergunta:**

* O algoritmo terminou?

**Ação:**

* Encerrar

---

# **DICA IMPORTANTE (ORGANIZAÇÃO DA SAÍDA)**

Para não deixar os resultados “grudados” na tela, use `\n` para pular linha.

### **Como usar:**

```c
printf("Resultado 1: %d\n", valor1);
printf("Resultado 2: %f\n", valor2);
```

### **Regra simples:**

* Sempre que quiser quebrar linha, use `\n` dentro das aspas

---

# **BLOCO SEPARADO (EXEMPLOS — NÃO COPIAR DIRETAMENTE)**

### **Leitura:**

```c
printf("Digite a idade: ");
scanf("%d", &idade);
```

### **Saída:**

```c
printf("Idade: %d\n", idade);
```
