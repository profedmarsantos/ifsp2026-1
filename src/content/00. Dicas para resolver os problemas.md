# **ROTEIRO PADRÃO PARA RESOLVER ALGORITMOS**

## **Ordem fixa (seguir sempre):**

1. Entender o problema
2. Definir variáveis
3. Ler dados
4. Processar
5. Mostrar resultado
6. Finalizar

---

# **ROTEIRO COMPLETO (COM PERGUNTAS GUIA)**

## **1. Entender o problema**

**Pergunta guia:**

* “O que eu preciso calcular?”

**O que fazer:**

* Ler o enunciado
* Identificar o(s) resultado(s)

---

## **2. Definir variáveis**

**Pergunta guia:**

* “Quais variáveis eu preciso usar?”

**O que fazer:**

* Criar variáveis para todos os valores
* Pode ter uma ou várias

---

### **Regra fixa (tipo de variável):**

Use **int** para:

* Idade
* Quantidade
* Contagem

```c
int idade;
int quantidade;
```

---

Use **float** para:

* Nota
* Média
* Dinheiro
* Altura

```c
float nota1;
float nota2;
float media;
```

---

## **3. Ler dados**

**Pergunta guia:**

* “Quais valores o usuário precisa digitar?”

**O que fazer:**

* Ler todos os dados necessários

---

### **Regra fixa (sempre seguir este padrão):**

**1. Mostrar mensagem**

```c
printf("Digite ...: ");
```

**2. Ler valor**

```c
scanf("%d", &variavel);   // int
scanf("%f", &variavel);   // float
```

---

### **Regras importantes:**

* Sempre mostrar uma mensagem antes
* Sempre usar `&`
* `%d` → int
* `%f` → float
* Repetir para cada variável

---

### **Exemplo:**

```c
printf("Digite a primeira nota: ");
scanf("%f", &nota1);

printf("Digite a segunda nota: ");
scanf("%f", &nota2);

printf("Digite a quantidade: ");
scanf("%d", &quantidade);
```

---

## **4. Processar**

**Pergunta guia:**

* “Que cálculo eu preciso fazer?”

**O que fazer:**

* Fazer um ou mais cálculos

```c
media = (nota1 + nota2) / 2;
total = quantidade * preco;
```

---

## **5. Mostrar resultado**

**Pergunta guia:**

* “O que eu preciso mostrar para o usuário?”

**O que fazer:**

* Mostrar todos os resultados

---

### **Regra fixa:**

```c
printf("Texto: %d", variavel);   // int
printf("Texto: %f", variavel);   // float
```

---

### **Exemplo:**

```c
printf("Media: %f", media);
printf("Total: %d", total);
```

---

## **6. Finalizar**

**Pergunta guia:**

* “O algoritmo terminou?”

**O que fazer:**

* Encerrar o programa
