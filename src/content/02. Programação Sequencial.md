### 1. Fundamentação Teórica: O Paradigma Sequencial

#### 1.1 Definição e Características
A programação sequencial é o paradigma no qual as instruções são executadas estritamente na ordem em que aparecem no código fonte.
*   **Linearidade:** A execução inicia na primeira linha (dentro do `main`) e prossegue linha por linha até encontrar o `return 0;`.
*   **Determinismo:** Dado um mesmo conjunto de dados de entrada, um algoritmo puramente sequencial produzirá sempre exatamente o mesmo resultado.
*   **Ausência de Desvios:** Neste estágio, não utilizamos estruturas que alteram o fluxo (como `if`, `else`, `while`, `for`). Todas as linhas são executadas obrigatoriamente.

#### 1.2 Estrutura Padrão de um Programa Sequencial em C
Todo programa sequencial desenvolvido nesta disciplina seguirá rigorosamente três etapas lógicas, mesmo que visualmente misturadas no código:

1.  **Declaração de Variáveis:** Reserva de espaço na memória para armazenar os dados. Deve ocorrer antes de qualquer uso das variáveis.
2.  **Entrada de Dados (Input):** Coleta de informações do usuário (via teclado) para alimentar as variáveis. Sem isso, o programa trabalha com "lixo" de memória ou valores fixos.
3.  **Processamento:** Aplicação de expressões aritméticas e lógicas sobre os dados de entrada para gerar novos valores.
4.  **Saída de Dados (Output):** Exibição dos resultados processados para o usuário.

**Fluxograma Conceitual:**
```text
[INÍCIO]
   |
   v
[Declaração de Variáveis]
   |
   v
[Entrada de Dados (leia)]
   |
   v
[Processamento (fórmulas)]
   |
   v
[Saída de Dados (escreva)]
   |
   v
[FIM]
```

#### 1.3 Boas Práticas na Programação Sequencial
*   **Inicialização:** Sempre atribua um valor às variáveis (via `scanf` ou atribuição direta) antes de usá-las em cálculos. Usar uma variável não inicializada gera comportamento indefinido.
*   **Ordem das Operações:** Certifique-se de que todos os dados necessários para uma fórmula foram lidos *antes* da linha que contém o cálculo.
    *   *Errado:* Calcular `area = base * altura` antes de ler `base` e `altura`.
    *   *Certo:* Ler `base`, ler `altura`, depois calcular `area`.
*   **Clareza na Saída:** As mensagens de `printf` devem ser descritivas, indicando unidades de medida (R$, km, kg, %) para que o usuário interprete corretamente o resultado.

#### 2.1 Estudo de Caso 1: Cálculo Salarial com Desconto (Baseado no Exercício 3)

**Problema:**
Realizar o cálculo salarial de um trabalhador que recebe por hora. O programa deve ler o valor da hora trabalhada, o número de horas trabalhadas e o percentual de desconto do INSS. Deve mostrar o salário bruto, o valor do desconto e o salário líquido.

**Descrição em Linguagem Natural:**
O sistema solicita ao usuário o valor ganho por hora, a quantidade de horas trabalhadas no mês e a porcentagem de desconto do INSS. Primeiro, calcula-se o salário bruto multiplicando valor da hora pelas horas trabalhadas. Em seguida, calcula-se o valor do desconto aplicando a porcentagem sobre o bruto. Por fim, subtrai-se o desconto do bruto para obter o líquido. Todos os três valores são exibidos.

**Pseudocódigo Estruturado:**
```text
VARIÁVEIS
    real valor_hora, horas_trabalhadas, percentual_inss
    real salario_bruto, valor_desconto, salario_liquido

INICIO
    // Entrada de Dados
    escreva("Digite o valor da hora trabalhada: ")
    leia(valor_hora)
    
    escreva("Digite o numero de horas trabalhadas: ")
    leia(horas_trabalhadas)
    
    escreva("Digite o percentual de desconto do INSS: ")
    leia(percentual_inss)

    // Processamento
    salario_bruto = valor_hora * horas_trabalhadas
    valor_desconto = salario_bruto * (percentual_inss / 100.0)
    salario_liquido = salario_bruto - valor_desconto

    // Saída de Dados
    escreva("Salario Bruto: R$ ", salario_bruto)
    escreva("Valor do Desconto (INSS): R$ ", valor_desconto)
    escreva("Salario Liquido a Receber: R$ ", salario_liquido)
FIM
```

**Código em C:**
```c
#include <stdio.h>

int main() {
    // Declaração de Variáveis
    float valor_hora, horas_trabalhadas, percentual_inss;
    float salario_bruto, valor_desconto, salario_liquido;

    // Entrada de Dados
    printf("Digite o valor da hora trabalhada: ");
    scanf("%f", &valor_hora);

    printf("Digite o numero de horas trabalhadas: ");
    scanf("%f", &horas_trabalhadas);

    printf("Digite o percentual de desconto do INSS: ");
    scanf("%f", &percentual_inss);

    // Processamento (Sequência Lógica de Cálculos)
    // Nota: A ordem aqui é crucial. Não podemos calcular o desconto antes do bruto.
    salario_bruto = valor_hora * horas_trabalhadas;
    valor_desconto = salario_bruto * (percentual_inss / 100.0);
    salario_liquido = salario_bruto - valor_desconto;

    // Saída de Dados
    printf("\n--- Demonstrativo de Pagamento ---\n");
    printf("Salario Bruto: R$ %.2f\n", salario_bruto);
    printf("Valor do Desconto (INSS): R$ %.2f\n", valor_desconto);
    printf("Salario Liquido a Receber: R$ %.2f\n", salario_liquido);

    return 0;
}
```

**Análise de Erros Comuns neste Cenário:**
1.  **Ordem Invertida:** Tentar calcular `salario_bruto` antes de executar os `scanf`. O programa compila, mas o resultado será incorreto (geralmente zero ou um número aleatório).
2.  **Erro de Porcentagem:** Esquecer de dividir por 100 na hora de calcular o desconto (`salario_bruto * percentual_inss` estaria errado, pois transformaria 10% em 1000% do valor).
3.  **Tipo de Dado:** Usar `int` para moeda ou horas fracionadas, causando perda de precisão nos centavos ou minutos.

#### 2.2 Estudo de Caso 2: Juros Simples (Baseado no Exercício 11)

**Problema:**
Calcular o juro simples produzido por um capital aplicado. O programa deve ler o Capital (X), a Taxa de Juros mensal em porcentagem (Y) e o tempo em meses (W). Deve mostrar o capital, o juro calculado e o montante final.

**Fórmula Matemática:**
$J = C \cdot i \cdot t$ (onde $i$ é a taxa unitária, ou seja, porcentagem / 100)
$M = C + J$

**Descrição em Linguagem Natural:**
O usuário informa o valor inicial investido, a taxa de juros mensal (%) e a duração da aplicação em meses. O programa converte a taxa porcentual para decimal, calcula o valor dos juros multiplicando capital, taxa e tempo. Depois, soma os juros ao capital original para achar o montante. Exibe os três valores.

**Pseudocódigo Estruturado:**
```text
VARIÁVEIS
    real capital, taxa_percentual, tempo_meses
    real taxa_unitaria, juros, montante

INICIO
    // Entrada
    escreva("Digite o Capital (R$): ")
    leia(capital)
    
    escreva("Digite a Taxa de Juros mensal (%): ")
    leia(taxa_percentual)
    
    escreva("Digite o Tempo de aplicacao (meses): ")
    leia(tempo_meses)

    // Processamento
    taxa_unitaria = taxa_percentual / 100.0
    juros = capital * taxa_unitaria * tempo_meses
    montante = capital + juros

    // Saída
    escreva("Capital Inicial: R$ ", capital)
    escreva("Juros Gerados: R$ ", juros)
    escreva("Montante Final: R$ ", montante)
FIM
```

**Código em C:**
```c
#include <stdio.h>

int main() {
    // Declaração
    float capital, taxa_percentual, tempo_meses;
    float taxa_unitaria, juros, montante;

    // Entrada
    printf("Digite o Capital (R$): ");
    scanf("%f", &capital);

    printf("Digite a Taxa de Juros mensal (%%): ");
    scanf("%f", &taxa_percentual);

    printf("Digite o Tempo de aplicacao (meses): ");
    scanf("%f", &tempo_meses);

    // Processamento
    // Passo intermediário explícito para clareza lógica
    taxa_unitaria = taxa_percentual / 100.0; 
    
    // Fórmula de Juros Simples: J = C * i * t
    juros = capital * taxa_unitaria * tempo_meses;
    
    // Fórmula do Montante: M = C + J
    montante = capital + juros;

    // Saída
    printf("\n--- Resultado da Aplicação ---\n");
    printf("Capital Inicial: R$ %.2f\n", capital);
    printf("Juros Gerados: R$ %.2f\n", juros);
    printf("Montante Final: R$ %.2f\n", montante);

    return 0;
}
```
