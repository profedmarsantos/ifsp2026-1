# VETORES E MATRIZES: ARMAZENANDO E ORGANIZANDO DADOS

## 1. PROBLEMA INICIAL (DESENVOLVIMENTO RÁPIDO)
#### (1) Problema
Desenvolva um programa que leia **5 notas** de um aluno e, ao final, calcule a média. Com base no valor da média, exiba a situação do aluno:
- **Aprovado:** média maior ou igual a 6,0
- **Recuperação:** média maior ou igual a 4,0 e menor que 6,0
- **Reprovado:** média menor que 4,0

#### (2) Instruções
1. **Preparação de Variáveis**
   1.1. Declare **5 variáveis** do tipo `float` para armazenar cada nota (`nota1`, `nota2`, `nota3`, `nota4`, `nota5`).
   1.2. Declare uma variável `float` para a `soma` e outra para a `media`.
   1.3. **Inicialize** `soma` com `0.0` antes de qualquer cálculo.
2. **Entrada de Dados**
   2.1. Solicite e leia as 5 notas separadamente usando `scanf`.
3. **Processamento**
   3.1. Some todas as notas e armazene em `soma`.
   3.2. Divida `soma` por `5.0` e armazene o resultado em `media`.
4. **Saída e Decisão**
   4.1. Exiba a média formatada com duas casas decimais.
   4.2. Utilize `if`, `else if` e `else` para comparar a `media` e imprimir a situação correspondente.

#### (3) Exemplo de como deve ficar
```
Digite a nota 1: 7.5
Digite a nota 2: 6.0
Digite a nota 3: 5.5
Digite a nota 4: 8.0
Digite a nota 5: 7.0
Media final: 6.80
Situacao: Aprovado
```

#### (4) Solução Imediata
```c
#include <stdio.h>

int main() {
    float nota1, nota2, nota3, nota4, nota5;
    float soma = 0.0, media;

    printf("Digite a nota 1: "); scanf("%f", &nota1);
    printf("Digite a nota 2: "); scanf("%f", &nota2);
    printf("Digite a nota 3: "); scanf("%f", &nota3);
    printf("Digite a nota 4: "); scanf("%f", &nota4);
    printf("Digite a nota 5: "); scanf("%f", &nota5);

    soma = nota1 + nota2 + nota3 + nota4 + nota5;
    media = soma / 5.0;

    printf("Media final: %.2f\n", media);

    if (media >= 6.0) {
        printf("Situacao: Aprovado\n");
    } else if (media >= 4.0) {
        printf("Situacao: Recuperacao\n");
    } else {
        printf("Situacao: Reprovado\n");
    }

    return 0;
}
```

## 2. A LIMITAÇÃO DAS VARIÁVEIS SIMPLES
O programa anterior funciona, mas apresenta **duas falhas estruturais graves** quando a quantidade de dados aumenta:
1. **Repetição manual:** Se forem 100 notas, serão necessárias 100 declarações e 100 comandos `scanf`.
2. **Impossibilidade de percorrer os dados:** Para exibir as notas na ordem inversa, encontrar a menor nota ou calcular estatísticas adicionais, você não tem um nome único que identifique o conjunto. Cada nota é uma variável isolada.

Matematicamente, trabalhamos com conjuntos indexados: `N = {n1, n2, n3, n4, n5}`. Na programação, replicamos esse conceito utilizando uma **única variável composta** que contém múltiplas posições acessíveis por um número inteiro.

## 3. CONCEITO DE VETORES E MATRIZES
### 3.1 Vetor (Unidimensional)
Um vetor é uma variável que organiza dados em **uma única dimensão linear**. Pense nele como **uma única linha ou uma única coluna** em uma planilha. Todas as posições armazenam o mesmo tipo de dado.

**Exemplo visual:** `[ | | | | | ]` → 5 espaços contíguos na memória, acessados por um único número (índice).

### 3.2 Matriz (Bidimensional)
Uma matriz organiza dados em **linhas e colunas**, exigindo **dois números** para localizar um elemento específico. A analogia mais direta é o **Microsoft Excel**:
- No Excel, uma célula é identificada por **Letra (Coluna) + Número (Linha)** (ex: `B3`).
- Na programação, substituímos as letras por números e invertemos a ordem padrão: usamos **`[linha][coluna]`**.
- A célula `B3` (2ª coluna, 3ª linha) corresponde, conceitualmente, a `matriz[2][1]` (considerando índice 0).

**Exemplo visual (3 linhas x 4 colunas):**
```
[ ][ ][ ][ ]  ← Linha 0
[ ][ ][ ][ ]  ← Linha 1
[ ][ ][ ][ ]  ← Linha 2
```

### 3.3 Relação entre Matriz Coluna e Vetor
Uma **matriz coluna** possui dimensão `N x 1`. Ela armazena os elementos verticalmente, mas, na memória do computador, os dados são guardados sequencialmente. Por isso, **um vetor em C é tratado como uma matriz linearizada**. A diferença é apenas sintática: o vetor usa **um índice**; a matriz usa **dois índices**. Ambos compartilham o mesmo princípio de acesso direto via cálculo de endereço de memória.

## 4. SINTAXE E INDEXAÇÃO EM C
Em C puro, a declaração segue a estrutura:
```c
tipo_de_dado nome_da_variavel[dimensao];
```
- **Dimensão:** Define a **quantidade fixa** de posições reservadas na memória.
- **Índice de posição:** **SEMPRE INICIA EM 0** e termina em `dimensão - 1`.
- **Acesso:** `nome_da_variavel[indice]`

**Exemplo:**
```c
int notas[5];
```
Reserva 5 espaços inteiros contíguos. As posições válidas são `notas[0]`, `notas[1]`, `notas[2]`, `notas[3]` e `notas[4]`.
**Regra crítica:** Acessar `notas[5]` ou `notas[-1]` viola os limites da memória reservada e causa **comportamento indefinido** (falhas silenciosas ou travamento do programa).

## 5. RESOLUÇÃO PASSO A PASSO: NOTAS COM VETOR
**Problema:** Ler 5 notas, armazená-las, exibir todas elas na ordem de entrada e calcular a média final.

**(1) Declaração Estruturada**
```c
#include <stdio.h>
#define TAM 5 // Define o tamanho fixo do vetor

int main() {
    float notas[TAM]; // Vetor de 5 posições
    float soma = 0.0; // Acumulador inicializado
    float media;
    int i;            // Variável de controle dos laços
```

**(2) Leitura com Laço**
```c
    // Percorre do índice 0 até 4 (TAM-1)
    for (i = 0; i < TAM; i++) {
        printf("Digite a nota %d: ", i + 1);
        scanf("%f", &notas[i]); // Armazena na posição atual
    }
```

**(3) Cálculo da Média**
```c
    // Segundo percurso para somar os valores preservados
    for (i = 0; i < TAM; i++) {
        soma = soma + notas[i];
    }
    media = soma / TAM;
```

**(4) Exibição dos Dados e Situação**
```c
    printf("\nNotas armazenadas:\n");
    for (i = 0; i < TAM; i++) {
        printf("Posicao %d: %.1f\n", i, notas[i]);
    }

    printf("Media final: %.2f\n", media);
    if (media >= 6.0) printf("Situacao: Aprovado\n");
    else if (media >= 4.0) printf("Situacao: Recuperacao\n");
    else printf("Situacao: Reprovado\n");

    return 0;
}
```
**Porquê antes do como:** O vetor **preserva os dados em memória**. Isso permite múltiplas passagens de leitura (leitura → cálculo → exibição) sem perder a informação original, algo impossível com variáveis simples reutilizadas.

### EXERCÍCIO 3: Busca por Valor Máximo e Valor Mínimo
#### (1) Problema
Crie um programa que preencha um **vetor de 10 números reais**, identifique o **maior valor** e o **menor valor**, junto com suas **posições originais (índice)**. Exiba ambos.

#### (2) Instruções
1. **Preparação e Leitura**
   1.1. Declare um vetor de `float` com tamanho **10**.
   1.2. Declare variáveis para `maior`, `menor`, `pos_maior` e `pos_menor`.
   1.3. Utilize um `for` para ler os 10 números do usuário.
2. **Inicialização Segura dos Comparadores**
   2.1. **Antes de qualquer comparação**, atribua `vetor[0]` às variáveis `maior` e `menor`.
   2.2. Atribua `0` às variáveis `pos_maior` e `pos_menor`.
   *(Esta etapa evita falhas quando todos os números são negativos ou positivos).*
3. **Varredura e Atualização**
   3.1. Crie um laço `for` iniciando no índice `1` até `9`.
   3.2. Dentro do laço, verifique se `vetor[i] > maior`. Se verdadeiro:
      3.2.1. Atualize `maior = vetor[i]`.

   3.3. Em seguida, verifique se `vetor[i] < menor`. Se verdadeiro:
      3.3.1. Atualize `menor = vetor[i]`.
      3.3.2. Atualize `pos_menor = i`.
4. **Exibição dos Extremos**
   4.1. Imprima o valor e a posição do maior.
   4.2. Imprima o valor e a posição do menor, formatando os números reais com duas casas decimais.

#### (3) Exemplo de como deve ficar
```

Maior valor: 9.10 na posicao 6
```

## 7. SOLUÇÕES DOS EXERCÍCIOS

### Solução - Exercício 1: Sequência Inversa
```c
#include <stdio.h>

#define TAM 6

int main() {
    int numeros[TAM];
    int i;

    // 1. Leitura sequencial do índice 0 até 5
    for (i = 0; i < TAM; i++) {
        printf("Digite o numero %d: ", i + 1);
        scanf("%d", &numeros[i]);
    }

    printf("Sequencia inversa: ");
    // 2. Percorre do último índice (TAM-1) até o primeiro (0)
    for (i = TAM - 1; i >= 0; i--) {
        printf("%d ", numeros[i]);
    }
    printf("\n");

    return 0;
}
```

### Solução - Exercício 2: Soma de Elementos em Vetor
```c
#include <stdio.h>

#define TAM 8

int main() {
    float numeros[TAM];
    float soma = 0.0;
    int i;

    // 1. Leitura e acumulação simultânea
    for (i = 0; i < TAM; i++) {
        printf("Digite o numero %d: ", i + 1);
        scanf("%f", &numeros[i]);
        
        // 2. Acumula o valor na variável soma
        soma = soma + numeros[i];
    }

    // 3. Exibição do resultado final
    printf("Soma total dos elementos: %.2f\n", soma);

    return 0;
}
```

### Solução - Exercício 3: Busca por Valor Máximo e Mínimo
```c
#include <stdio.h>

#define TAM 10

int main() {
    float valores[TAM];
    float maior, menor;
    int pos_maior, pos_menor;
    int i;

    // 1. Leitura dos 10 números
    printf("Digite %d numeros reais:\n", TAM);
    for (i = 0; i < TAM; i++) {
        scanf("%f", &valores[i]);
    }

    // 2. Inicialização segura com o primeiro elemento
    maior = valores[0];
    menor = valores[0];
    pos_maior = 0;
    pos_menor = 0;

    // 3. Varredura a partir do índice 1
    for (i = 1; i < TAM; i++) {
        if (valores[i] > maior) {
            maior = valores[i];
            pos_maior = i;
        }
        if (valores[i] < menor) {
            menor = valores[i];
            pos_menor = i;
        }
    }

    // 4. Exibição dos resultados
    printf("Maior valor: %.2f na posicao %d\n", maior, pos_maior);
    printf("Menor valor: %.2f na posicao %d\n", menor, pos_menor);

    return 0;
}
```
